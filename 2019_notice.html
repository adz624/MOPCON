Hi 親愛的會眾您好,

第八屆 MOPCON -- 行動科技年會即將在 10/19 及 10/20 登場囉，以下是活動的相關資訊，請您行前先行閱讀，並做好相應之準備，以加速您入場的程序。

活動地點
# 高雄國際會議中心 ICCK 3、4 樓 (高雄市鹽埕區中正四路 274 號 3、4 樓)
# 細酌牛飲駁二店 （高雄市鹽埕區大義街 2-6 號 (大義倉庫 C11-2 )）


活動時間
2019 / 10 / 19 (六) 08:30 -22:00
2019 / 10 / 20 (日) 08:30 - 18:00

報到方式
# 兩天報到時間皆為 08:30 - 09:00，地點於 ICCK 3 樓，兩天皆須報到喔！
# 晚宴開放報到時間為 18：00，地點於細酌牛飲駁二店門口喔！

我們採用的是 KKTIX 跨平台報名服務，您有三種方式可以成功完成報到手續：
# 手機下載 KKTIX App  並登入會員，使用 QR code 進行報到。
# 至 KKTIX 的「個人帳務清單」頁面中，將 QR code 列印出來進行報到。
# 將報名成功時寄送至您 Email 中的電子通知信中的 QR code 儲存在手機上或是列印出來進行報到。


貼心小提醒
# 因為會場冷氣為中央空調，請您視自身偏好調整穿著或攜帶一件薄外套以備不時之需。
# 晚宴現場提供啤酒暢飲，請您視自身健康情況取用，若您自行開車，請事先安排代駕或鄰近住宿，喝酒不開車。

收據領取方式
# 若您需要開立收據，請於議程日當天至大會服務台（非報到處）進行申請與領取，活動結束後不接受收據申請與補開之服務。

用餐資訊
# 不提供午餐，可參考大會準備的美食地圖資訊，深入體驗鹽埕美食的風味。
# 兩日下午提供茶點及飲料，敬請自行攜帶環保杯一起愛地球。

交通資訊
高鐵：搭乘高鐵至高鐵左營站，轉搭高雄捷運至紅線捷運美麗島站，再轉乘橘線在捷運鹽埕埔站下車，往 2 號出口，步行約 3 至 5 分鐘即可抵達會場。

捷運：橘線捷運鹽埕埔站 2 號出口，步行約 3 至 5 分鐘即可抵達會場。

火車：搭乘火車至高雄火車站，轉搭高雄捷運至紅線捷運美麗島站，再轉乘橘線在捷運鹽埕埔站下車，往 2 號出口，步行約 3 至 5 分鐘即可抵達會場。

自行開車：鼎金民族匝道施工公告，若您有需要行走鼎金民族匝道上國道十號、或南下國道一號，請記得改道「文自、大中路匝道」。

市區公車：歷史博物館 0 南 (單邊停靠)、0 北 (單邊停靠)、11、25、33、56、60、76、77、82、建國幹線、91、214、219、248。
