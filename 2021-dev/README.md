# MOPCON 2021

## 環境變數設定

複製 `.env.example` 至 `.env`。

```
cp .env.example .env
```

至 `.env` 設定環境變數，請將以下設定依照開發環境修改。

```dotenv
BASE_URL=https://mopcon.org
```

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# generate static project
$ npm run generate
```
